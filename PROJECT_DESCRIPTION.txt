# SynthAI - Project Description

## Overview
SynthAI is a smart agricultural intelligence platform that generates synthetic agricultural datasets using CTGAN (Conditional Tabular Generative Adversarial Network). The platform serves three user roles: Farmers, Companies, and Administrators. It features a subscription-based payment system with tiered access to dataset sizes and prediction limits.

## Architecture
- **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript (no frameworks)
- **Backend**: Python with Streamlit (legacy), FastAPI (optional)
- **Storage**: localStorage for user data, sessions, and generation history
- **Serving**: Python HTTP server (`python -m http.server 8080`)

## User Roles

### Farmer
- Crop-to-Conditions: Get optimal growing conditions for a selected crop
- Conditions-to-Crop: Recommend crops based on soil/environmental inputs
- Limited predictions per day based on plan (Free: 5, Pro: 50, Premium: Unlimited)
- Prediction history tracking

### Company
- Generate synthetic datasets (Crop, Soil, Weather types)
- Fixed dataset sizes: Medium (1,000), Large (10,000), Big Data (100,000)
- Dataset viewer with pagination, statistics, correlation heatmap
- CSV export functionality
- Cannot modify row counts (enforced by service layer)

### Admin
- System overview dashboard (users, predictions, datasets)
- Train/retrain CTGAN models
- Data validation (compare real vs synthetic data quality)
- User management (view, activate/deactivate accounts)

## Subscription Plans

### Free ($0)
- 5 predictions/day (Farmer)
- Medium datasets only (1,000 rows)
- 10 history entries
- Crop dataset type only
- Basic analytics

### Pro ($9.99/month)
- 50 predictions/day (Farmer)
- Large datasets (10,000 rows)
- 100 history entries
- All dataset types (Crop, Soil, Weather)
- Priority support

### Premium ($29.99/month)
- Unlimited predictions
- Big Data generation (100,000 rows)
- Unlimited history
- All dataset types + custom
- Advanced analytics dashboard
- 24/7 priority support

## Services

### AuthService (`auth.js`)
- User registration and login
- Session management (sessionStorage)
- Role-based access control
- Prediction attempt tracking and limits
- User status management (active/inactive)
- Default admin account: `admin / admin123`

### PaymentService (`payment.js`)
- Plan management (Free, Pro, Premium)
- Simulated payment processing
- Subscription expiration tracking (30-day cycles)
- Feature access validation by plan
- Payment history tracking

### DataService (`data.js`)
- Fixed-size dataset configuration
- Dataset type generators (Crop, Soil, Weather)
- Preview generation (first 50 rows)
- Statistics computation (mean, std, min, max, median)
- Correlation matrix computation
- CSV export with BOM for Excel compatibility

### HistoryService (`history.js`)
- Prediction entry tracking
- Search by crop, filter by date
- Export history to CSV
- Statistics aggregation

## Dataset Types

### Crop
Columns: N, P, K, temperature, humidity, ph, rainfall, label
Crops: rice, maize, cotton, jute, coffee, banana, mango, grapes

### Soil
Columns: N, P, K, organic_matter, moisture, texture, depth_cm
Textures: sandy, clay, loam, silt, peat

### Weather
Columns: temperature, humidity, precipitation, wind_speed, pressure, cloud_cover

## UI/UX Design
- Green agricultural color theme (#1B5E20, #2E7D32, #66BB6A)
- Clean, modern SaaS layout
- Fixed sidebar with SVG navigation icons
- Responsive card-based design
- Status messages (success, warning, error)
- Pagination for data tables
- Correlation heatmap with color-coded cells

## Security Features
- Session-based authentication
- Role-based page access
- Account activation/deactivation
- Failed attempt tracking (though passwords are plaintext in localStorage - suitable for demo only)

## Known Limitations
- Passwords stored in plaintext (demo/prototype only)
- Payment processing is simulated (no real gateway)
- Data generation uses random sampling (not actual CTGAN output in frontend)
- localStorage has ~5-10MB limit (unsuitable for full 100K row datasets)
- No backend API integration yet (planned for future)

## Setup Instructions
1. Ensure Python 3.x is installed
2. Navigate to project root: `cd SynthAI`
3. Start HTTP server: `python -m http.server 8080`
4. Open browser: `http://localhost:8080/frontend/pages/login.html`
5. Login with default admin: `admin / admin123`
6. Or register a new user
