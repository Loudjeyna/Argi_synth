# AGENTS.md

This repository contains only a CSV file (`Crop_recommendation.csv`) with crop recommendation data.
There are no source code files, tests, or build scripts.

The CSV has the following columns: N, P, K, temperature, humidity, ph, rainfall, label.
- 2200 data rows plus header
- 22 unique crop labels (apple, banana, blackgram, chickpea, coconut, coffee, cotton, grapes, jute, kidneybeans, lentil, maize, mango, mothbeans, mungbean, muskmelon, orange, papaya, pigeonpeas, pomegranate, rice, watermelon)
- N: Nitrogen ratio in soil
- P: Phosphorus ratio in soil  
- K: Potassium ratio in soil
- temperature: Temperature in Celsius
- humidity: Relative humidity in %
- ph: pH value of soil
- rainfall: Rainfall in mm
- label: Crop type

No special setup or commands are needed to work with this data.
